MOGL
=======
MOGL is a lightweight & high performance **MO**bile web**GL** library.

**Get MoGL**

* [Releases](https://github.com/projectBS/MoGL/releases)
* CDN : https://cdnjs.com/libraries/mogl
* Bower : `$ bower install mogl`
